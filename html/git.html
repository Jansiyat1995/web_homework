Git

Создание репозитория

  • git init - создать новый репозиторий
  • git clone url - скопировать существующий репозиторий

Работа с файлами

  • git add <файл> - добавить файл в подготовку
  • git add . - добавить все файлы
  • git commit -m "текст" - сохранить изменения
  • git status - посмотреть статус файлов

Ветки

  • git branch - показать ветки
  • git branch <имя> - создать ветку
  • git checkout <ветка> - переключиться на ветку
  • git checkout -b <имя> - создать и переключиться
  • git merge <ветка> - объединить ветки

Работа с удаленным репозиторием

  • git push - отправить изменения на сервер
  • git pull - скачать изменения с сервера
  • git fetch - проверить обновления

Отмена действий

  • git restore <файл> - отменить изменения
  • git restore --staged <файл> - убрать из подготовки
  • git reset --hard - откатить все до последнего коммита

История

  • git log - посмотреть историю
  • git log --oneline - краткая история
  • git diff - посмотреть изменения

Полезные команды

  • git stash - временно сохранить изменения
  • git stash pop - вернуть сохраненное
  • git remote -v - показать подключенные репозитории
  • git rm <файл> - удалить файл из Git
